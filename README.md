# PHP MVC Framework

This is a custom PHP MVC-style framework
built from scratch for learning and full control
over backend architecture.

## What it does right now

- Custom routing system
- Controller-based structure
- View rendering system
- Basic autoloading
- Organized MVC folders

## Project Structure

- app/controllers -> Handels request logic
- app/core -> Core framework (Router, Controller, Autoloader)
- app/models -> Data layer (not fully implemented yet)
- app/views -> UI templates
- config -> Configuration files
- public -> Web entry point (index.php)
- storage -> Logs and uploaded files

## Current Goal

Build a secure, modular framework with:

- Authentication system
- Middleware layer
- Database abstraction
- Clean routing system
